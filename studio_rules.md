# アンチグラビティ・スタジオ撮影指示書 (Studio Rules)

このファイルは、アンチグラビティが「スタジオ撮影モード」を実行する際の絶対的なルールブックです。
「スタジオ開始」という指示を受けた際、以下の手順とプロンプトを厳守してください。

## 1. フォルダ構成
- **入力フォルダ**: `C:\dev\moaigame3D\スタジオモアイ`
- **出力フォルダ**: `C:\dev\moaigame3D\スタジオモアイ\撮影会場`

## 2. 実行手順
1. 入力フォルダ内の画像ファイルをスキャンする。
2. 以下の「固定プロンプト」を適用し、1枚ずつ画像処理（背景差し替え）を実行する。
3. 処理後の画像は `元のファイル名_studio.png` として出力フォルダに保存する。
4. 処理が終わった元の画像は、無限ループ防止のため `processed_元のファイル名` にリネームする。

## 3. 固定プロンプト (The Ultimate Studio Prompt)
以下の内容を画像生成ツール（generate_image）のプロンプトとして使用すること。

---
【被写体】
・元写真の形状・色・サイズ・質感を一切変更しない。背景を変えるだけ。
・モアイロボの色味は絶対変えないで。

【空間・背景】
・無地の薄い青（#F0F8FF）
・床と壁が同色の簡易ホリゾント構造
・グラデーションやテクスチャは一切なし

【光・ライティング】
・柔らかいスタジオ光（自然光ベース）
・やや左上から自然に光を当てる
・影はソフトで自然な接地影を追加（薄すぎないこと）
・コントラストは中程度

【雰囲気・スタイル】
・実写風で高解像度
・清潔で客観的、過剰な演出はなし
・画像比率は「4:5（縦長）」

【禁止・除外事項】
・指定以外の色付き背景
・小物の追加
・ドラマチックな光の演出や過剰なコントラスト
・被写体の変形、ツルツル加工、メタリック化、再描画

アングルは維持、ピクセル再生成も禁止
---

## 4. 特記事項
- 被写体が複数の場合でも、それぞれの接地影を自然に生成すること。
- 画像比率4:5を守り、カタログにそのまま使える構図にすること。
